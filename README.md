# Personal Website
Repository of my Personal [website](https://www.julianbeck.com)

Written in Vue.js using Vue CLI 3
## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```
